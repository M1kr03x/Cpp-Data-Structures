# Data Structures \& Algorithms Library (C++)

##### Коллекция классических структур данных и алгоритмов, реализованных на современном C++ (стандарты 17/20). 

# 

# Технологический стек

Язык: C++17/20

Система сборки: CMake 3.15+

Управление памятью: Smart Pointers (unique\_ptr, shared\_ptr), RAII.





# Структура проекта

include/ — Заголовочные файлы (.hpp) с шаблонными классами.

src/ — Примеры использования и реализация нешаблонных методов.
